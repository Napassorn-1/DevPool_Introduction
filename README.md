# 🚀 DevPool Introduction

โปรเจกต์นี้เป็นเว็บแนะนำตัว (Personal Portfolio) ที่จัดทำเพื่อเข้าร่วมโครงการ **DevPool 2569 #1 Round Mini Build Challenge**

เว็บไซต์นี้แสดงข้อมูลส่วนตัว งานที่รับผิดชอบ เหตุผลที่สนใจเข้าร่วมโครงการ พร้อมฟีเจอร์เล็ก ๆ ที่เชื่อมต่อกับ API ภายนอกเพื่อฝึกการพัฒนาเว็บแบบ Full Stack เบื้องต้น

---

## ✨ Features

### 👤 Profile Section
- ข้อมูลส่วนตัว เช่น ชื่อ สังกัด ตำแหน่ง และอีเมล

### 💼 Responsibilities
- งานที่รับผิดชอบในปัจจุบัน

### 🎯 Motivation
- เหตุผลที่สนใจเข้าร่วมโครงการ DevPool

### 🌙 Dark Mode
- สามารถสลับโหมดมืด/สว่างได้ด้วย JavaScript
- ใช้ DOM manipulation (`classList.toggle`)

### 💬 Quote of the Day (API Integration)
- ดึงข้อมูลคำคมจาก API ภายนอกด้วย `fetch()`
- ตัวอย่าง API:
  - https://dummyjson.com/quotes/random
- แสดงผลแบบ auto บนหน้าเว็บ
